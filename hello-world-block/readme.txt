A super-simple block that prints "Hello World" in the editor and on the frontend.

== Installation ==
1. Upload the plugin zip via Plugins → Add New → Upload Plugin.
2. Activate the plugin.
3. In the block editor, search for "Hello World" (category: Widgets).
